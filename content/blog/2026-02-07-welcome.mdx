This portfolio is a Next.js app on Netlify. Jenkins on Kubernetes runs CI. The [hub-and-spoke observability stack](/projects/central-observability-hub-stack) and Argo CD setup are the same infrastructure I write about here.

The blog records how those systems were built: tradeoffs, failures, and the parts that were boring but load-bearing.

## What You'll See Here

- **Build notes** on projects like the [Hybrid Cloud GitOps Control Plane](/projects/hybrid-cloud-gitops-control-plane) and [Rocket.Chat on AKS](/projects/rocketchat-kubernetes-enterprise)
- **Platform work**: Kubernetes, GitOps, observability, CI/CD from running workloads on OCI and Azure
- **Frontend work**: performance, accessibility, and layout on this site ([Systems](/systems) has the architecture map)

## How Posts Are Structured

Each post covers:

1. **Context**: the problem and constraints
2. **Approach**: what shipped and what did not
3. **Takeaway**: a checklist, snippet, or pattern you can reuse

If a post does not end with something concrete, it should not go up.
