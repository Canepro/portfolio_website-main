## Why These Changes?

- **Link vs anchor tags**: Next.js requires using the `<Link>` component for internal navigation to enable client-side routing and better performance
- **useCallback**: Prevents unnecessary re-creation of functions on every render, which was causing the useEffect to run more than necessary
- **useEffect dependencies**: React's exhaustive-deps rule requires all values used inside useEffect to be listed in the dependency array
